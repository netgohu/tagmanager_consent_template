# Description

This tag ensures that the consent mode functions properly by telling Google what data it can access and use via tags and pixels.

It is also compatible with GTMâ€™s consent feature.

# Instruction

The template provides the opportunity to set the default state of the consent what is given to Google. The consent state will be updated after the user has accepted or denied through the consent controlling page. The users choice is stored in the relevant cookie and the consent is given accordingly.

This template is intended for webshops operating with the NetShop V3 engine.

Please read the official vendor documentation for more information about consent states and the different options.
